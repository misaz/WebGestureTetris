Gesture Tetris

Play with gesture sensor!

  1. 1.Assemble the MAX25405 Gesture Sensor Evaluation Kit by connecting Shield and Microcontroller boards together.
  2. 2.Connect Sensor Board to the Kit using Ribbon cable.
  3. 3.Connect Power Adapter to the Shield.
  4. 4.Connect Kit to the computer using USB cable.
  5. 5.Press or press C key.
  6. 6.Select device and confirm dialog.
  7. 7.Swipe hand up in the front of sensor to start the game.
